FEM1D to MATLAB - July,31-2013

1. FEM1D in fortran has been converted to MATLAB files.
2. A_Main.m is the main program
3. All subroutines required to run the program is given in "Subroutines" Directory
4. A path has been set within the A_Main.m program to access the files in "Subroutines" and "Inputfiles" directories
5. To run the program
	A. Run A_Main.m and provide the necessary input and output files in the dialog box
	B. Inputfiles can either be in the directory of A_Main.m or in InputFiles directory
6. Verification
	A. The program has been verified for the input files given in "InputFiles" directory
	B. The results are given in "OutputFiles" directory
